
# Инструкция по генерации статей для EmojiVerse

## 1. Подготовка (один раз)
Убедитесь, что в корневой папке проекта (рядом с `package.json`) есть файл `.env`.
Внутри него должен быть ваш ключ:
```
API_KEY=AIzaSy...
```

## 2. Создание статьи
1. Создайте файл `input.txt` в корне проекта.
2. Вставьте в него текст статьи (просто текст, можно скопировать из Word).
3. Сохраните файл.

## 3. Запуск генератора
1. Откройте терминал (в VS Code: Terminal -> New Terminal).
2. Введите команду:
   ```bash
   npm run gen-post nazvanie-statyi-na-angliyskom
   ```
   *Пример: `npm run gen-post history-of-emoji`*

## 4. Что происходит дальше?
Скрипт автоматически:
1. Прочитает ваш текст.
2. Переведет его на 13 языков.
3. Создаст папку `data/articles/nazvanie-statyi`.
4. Создаст правильные TypeScript файлы.
5. **Безопасно** добавит статью в список на сайте (`data/blogPosts.ts`).

Вам останется только запустить сайт (`npm run dev`) и проверить результат!
